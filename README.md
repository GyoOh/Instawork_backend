## General info

This project contains the back-end code for Instawork's home assignment

## Technologies

Project is created with:
- express version: 4.18.2
- cors version: 2.8.5

## Setup

To run this project, install it locally using npm:

```
$ npm install
$ npm i nodemon
$ npm start or npm run dev
```
